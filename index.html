software livre

o que é:

Um software é considerado livre quando seu código-fonte está liberado segundo uma licença que dá mais liberdade de uso ao usuário que faz uso do programa.

para que serve:

O principal objetivo do software livre é deixar que os usuários realizem cópias, alterações, executá-los e até mesmo customizá-los da maneira que desejarem, sendo assim, ele também possui sua vantagem e desvantagem que vamos falar abaixo, confira algumas delas.

  • Customização:

    você tem total liberdade para realizar customizações. ao utilizar a programação é possível deixar o software da maneira que desejar, além disso adicionar novas funcionalidades fazendo com que você possa ganhar mais agilidade em seu dia a dia. Um bom exemplo é o Android, que se trata de um software livre, porém as grandes empresas pegam esse código e customizam cada um para a sua marca.

  • Custo-benefício:

    Boa parte dos software livres também são gratuitos. Uma pequena parte é paga, porém, por dar acesso ao código-fonte, ainda pode valer mais a pena que um proprietário.

  • Segurança:

    As pessoas que adquirem esses softwares estão focadas em aperfeiçoar cada vez mais e customizar como desejam. Como se trata de um código aberto, isso permite que possamos ver cada item que está sendo instalado na máquina. As licenças estão sempre sendo verificadas e testadas pela equipe de segurança.

  • Não depender de fornecedores:

    muitas vezes, quando adquirimos um software, ficamos dependendo de novas atualizações de softwares. Com o software livre, é possível adicionar novas funcionalidades quando desejar, além de ser possível implementar apenas as que atendem às suas necessidades.

curiosidade:

Para o software ser considerado livre, ele tem algumas liberdades que devem ser atendidas, sendo elas:

  • Liberdade 1:

    Permitir que a pessoa usuária execute o software para qualquer propósito;

  • Liberdade 2:

    Permitir que a pessoa usuária tenha livre acesso para que possa estudar e até mesmo customizar de acordo com sua preferência, atendendo às suas necessidades;

  • Liberdade 3:

    Permitir que a pessoa usuária possa compartilhar o software para outras pessoas para que elas utilizem também;

  • Liberdade 4:

    Permitir que a pessoa usuária possa realizar uma melhoria no software para depois entregar para outras pessoas com o intuito de que elas possam se beneficiar da melhor forma;

sofware propietario

o que é:

Um software proprietário, inverso do software livre, não pode ser copiado, modificado e até mesmo redistribuído. Essas opções acabam sendo bloqueadas por quem criou.

para que serve:

O software proprietário, privativo ou não livre, é um software para computadores que é licenciado com direitos exclusivos para o produtor. Conforme o local de distribuição do software, este pode ser abrangido por patentes.

curiosidade:

Como o código-fonte do software proprietário não está disponível para os usuários finais, há uma falta de transparência em relação ao funcionamento interno do software. Isso pode resultar em uma sensação de falta de controle sobre as operações e funcionalidades do sistema.

tipos de licença dos softwares:

O usuário geralmente precisa adquirir uma licença para usar o software proprietário e, muitas vezes, está sujeito a restrição quanto ao número de instalações ou dispositivos em que o software pode ser executado.

Já o software livre é distribuído com uma licença que concede aos usuários a liberdade de usar, estudar, modificar e distribuir o software. As licenças de software livre, como a General Public License (GPL), garantem essas liberdades

.

O que é Copyleft:

O copyleft é uma extensão das liberdades citadas acima, fazendo uma alusão e, ao mesmo tempo, uma sátira à lei de direitos autorais, o copyright. Assim como o software livre, ela permite que as pessoas tenham acesso ao conteúdo, porém citando a fonte desenvolvedora sempre que utilizarem.

O que é Copyright:

é o direito exclusivo do autor de reproduzir sua obra (literária, artística ou científica). Na maioria dos casos, a palavra copyright é acompanhada pela frase “todos os direitos reservados”, o que aponta que aquela obra está protegida por lei.

Creative Commons

é um tipo de licenciamento que permite que designers, cinegrafistas, escritores e desenvolvedores web usem conteúdo sem pagar royalties por ele. De livros a fotos, vídeo a som, há uma variedade de tipos de materiais que podem ser usados, desde que atribuídos adequadamente.
